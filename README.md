# Bundesliga

### Starting the .venv

First to run the packages in this repo, run the following line:

`python -m venv .venv`

Then run

`./.venv/Scripts/Activate.ps1`

### Install packages

Run the following in the terminal:

`pip install -r requirements.txt`

Then run

`poetry install`

## Start running the project

Open the file called `main.ipynb`
